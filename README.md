# 📌 Project Overview

This project is a **custom WordPress theme** built as part of a WordPress Developer assignment.
The theme implements a responsive UI with dynamic WordPress features including:

* Dynamic navigation menu
* Custom header and footer
* Hero slider section
* Product grid layout
* Responsive design for mobile, tablet, and desktop

The theme was developed locally using **Local WP** and follows WordPress theme development standards.

# 💻 Local Development Setup

The project was developed locally using **Local WP**.

## Step 1: Install Local WP

Download Local WP:

https://localwp.com/

Install the software and launch it.

---

## Step 2: Create a Local WordPress Site

1. Open **Local WP**
2. Click **Create New Site**
3. Enter site name:

```
madebydesign
```

4. Choose **Preferred Settings**
5. Create WordPress admin credentials.

Example:

```
Username: admin
Password: admin
```

6. Click **Add Site**

Local WP will automatically install WordPress.

---

## Step 3: Access the WordPress Site

After installation:

Open the site from Local WP:

```
http://madebydesign.local
```

Open WordPress dashboard:

```
http://madebydesign.local/wp-admin
```

# 🎨 Theme Development Process

The custom theme was created manually inside the WordPress themes directory.

Theme path:

```
wp-content/themes/madebydesign-theme
```

Theme structure:

```
madebydesign-theme
│
├── style.css
├── index.php
├── header.php
├── footer.php
├── functions.php
├── front-page.php
├── page.php
└── assets
    ├── js
    
```

---

# ⚙ Theme Installation

To use this theme in WordPress:

1. Download or clone this repository.

2. Copy the theme folder to:

```
wp-content/themes/
```

Example:

```
wp-content/themes/madebydesign-theme
```

3. Login to WordPress Dashboard:

```
http://your-site/wp-admin
```

4. Go to:

```
Appearance → Themes
```

5. Activate **Made By Design Theme**
